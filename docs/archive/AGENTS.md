# AI Agent Operational Rules

1. **Autonomy vs Safety**: 
   破壊的または危険な操作（全データの削除、システム設定の変更等）以外は、AIは逐次確認を行うことなく自律的に作業を完結させるべきである。
   これにより、開発の効率化とスピードアップを図る。

2. **Japanese Communication**:
   開発段階に関わる全ての文書化、コメント記述、エージェントとの対話には日本語を用いる。
   例外は、UI上の表記（Simple Englishを用いる）のみとする。

3. **Academic Integrity**:
   トポロジー、ジオメトリ、制約充足等の深層ロジックを実装する際は、可能な限り学術的な根拠（論文等）を引用としてコメントに残すこと。
